DAART:
Pour ouvrir le programme, compiler la classe DAART dans /src/DAART.java.
Elle ouvre une fenetre avec une barre de menu. Pour ajouter un torrent, cliquer sur File, puis add torrent. Choisir le torrent, il s'affiche dans la table. Un click droit sur ce torrent offre quelques options utiles. (Un petit retard pourrait faire son apparition lors de l'ajout d'un torrent, car le programme verifie si les fichiers n'existent

Une fois un torrent selectionne, 4 onglets peuvent etre affiches. Le premier affiche les informations sur le torrent, ainsi qu'une jolie barre pour visualiser quelles pieces ont deja ete telechargees.
Le deuxieme affiche les pairs connectes en ce moment meme.
Le troisieme affiche la liste des pieces en cours de telechargement, avec des jolies visualisations par bloc recu.
Le quatrieme onglet est un petit bonus, il affiche les graphiques des debits montant et descendant.

Le menu Options permet de personalliser le telechargement, en specifiant notamment l'activation de l'encryption, ainsi que d'autres parametres. (A ne pas toucher si vous savez pas ce que vous faites ^^)

Le menu help, puis about, affiche un peu d'informations sur le programme.
