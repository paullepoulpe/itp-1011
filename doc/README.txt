DAART:
Pour ouvrir le programme, compiler la classe DAART dans /src/DAART.java.
Elle ouvre une fenetre avec une barre de menu. Pour ajouter un torrent, cliquer sur File, puis add torrent. Choisir le torrent, il s'affiche dans la table. Un click droit sur ce torrent offre quelques options utiles.
une fois un torrent selectionne, 4 onglets peuvent etre affiches. Le premier affiche les informations sur le torrent, ainsi qu'une jolie barre pour visualiser quelles pieces
